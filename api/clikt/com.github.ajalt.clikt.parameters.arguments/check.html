Check the final argument value and raise an error if it's not valid.

The validator is called with the final argument type (the output of transformAll), and should return false if the value is not valid. You can specify a message to include in the error output.

You can use validate for more complex checks.

Example:

val arg by argument().int().check("value must be even") { it % 2 == 0 }
