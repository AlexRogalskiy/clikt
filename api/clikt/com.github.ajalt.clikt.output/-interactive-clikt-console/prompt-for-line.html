open override fun promptForLine(prompt: String, hideInput: Boolean): String?

Show the prompt to the user, and return a line of their response.

This function will block until a line of input has been read.

Return

A line of user input, or null if an error occurred.

Parameters

prompt

The text to display to the user

hideInput

If true, the user's input should not be echoed to the screen. If the current console does not support hidden input, this argument may be ignored. Currently, this argument is ignored on JS and Native platforms.
